
# credit_card_fraud_detection


A machine learning based project that identifies suspicious transactions using data-driven patterns and predictive modelling approach.


**Description**

Credit Card Fraud Detection is a machine learning classification project aimed at detecting fraudulent transactions by analysing historical transaction patterns.
The model helps banks, fintech products and financial security teams identify & flag potential fraud cases before loss happens.
